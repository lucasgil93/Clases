Ejercicio 2

Crea una web que use tailwind a partir de un CDN. Según la documentación inicial (https://tailwindcss.com/docs/installation/play-cdn), esto se consigue añadiendo al < head> la siguiente línea. 

 script src="https://cdn.tailwindcss.com"

En tu web habrá una etiqueta < p id="parrafo">Hola caracola y un formulario con un textbox y dos botones: Añadir y Eliminar. Cada vez que se pulse Añadir, se añadirá al párrafo la clase de tailwind que el usuario haya indicado en el textbox. Cada vez que se pulse Eliminar, se eliminará del párrafo la clase de tailwind que el usuario haya indicado en el textbox.


Ejemplos de clases para probar: 

        p-2
m-5
bg-indigo-200
w-1/2
border-solid border-2 border-indigo-500 
drop-shadow-lg

Hola caracola
