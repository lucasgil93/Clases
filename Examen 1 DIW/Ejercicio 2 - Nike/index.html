Ejercicio

Trata de recrear la tarjeta de producto tal y como aparece en las imágenes de abajo.

Reglas:

  • No es obligatorio usar el HTML y CSS que se proporcionan. Si el alumno lo desea puede empezar el ejercicio desde cero.
  • Tanto el HTML como el CSS que se proporcionan pueden ser modificados.
  • El tipo de letra utilizado es JosefinSans (Ver la nota de abajo).
  • El ancho máximo de la tarjeta de producto es 60ch.
  • El cambio de layout se produce a los 60ch.
  • Los colores usados son #c9021b, black, blue, white. Es obligatorio usar variables CSS para los colores.
  • La imagen es responsiva y se adapta al ancho de la tarjeta.
  • No se necesita (y no se debe usar) Javascript.

Nota: Sólo se ha usado px indicar el radio y ancho de los bordes.

Nota: Para incluir en el CSS una tipografía descargada se puede usar el siguiente código.

    @font-face {
      font-family: JosefinSans;
      src: url(JosefinSans-VariableFont_wght.ttf);
    }
