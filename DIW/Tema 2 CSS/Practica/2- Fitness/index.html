Card de una aplicación de fitness

Trata de recrear la tarjeta de producto tal y como aparece en la imágen de abajo.

Reglas:

  • No es obligatorio usar el HTML y CSS que se proporcionan. Si el alumno lo desea puede empezar el ejercicio desde cero.
  • Tanto el HTML como el CSS que se proporcionan pueden ser modificados.
  • El tipo de letra utilizado es QuickSand (Ver la nota de abajo).
  • El ancho máximo de la tarjeta de producto 80ch y el alto es 70ch en modo móvil y 50ch en modo escritorio.
  • El cambio de layout se produce a los 700px (cambia el alto de la card y la imagen).
  • Los colores usados son #00ff62, white y rgba(0,0,0,0.25) (en la parte translúcida). Es obligatorio usar variables CSS para los colores.
  • Las imágenes son responsivas y se adaptan al ancho de la tarjeta.
  • No se necesita (y no se debe usar) Javascript.

Nota: Sólo se ha usado px al indicar el radio y ancho de los bordes y al usar el efecto blur.

Nota: Para incluir en el CSS una tipografía descargada se puede usar el siguiente código.

        @font-face {
          font-family: Quicksand;
          src: url(Quicksand-VariableFont_wght.ttf);
        }
